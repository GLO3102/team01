    Functionnality implemented :

    New and improved API :

    Our API was design using, of course, the one provided for the course. In addition to the functionnalities already available, we enhanced and added the following elements :

    • Separate calls for the search that return differents objects containing grouped results (movies, tv shows, actors or users for instance)
    • Automated calls to the YouTube API, providing the proper adress for the preview videos directly in the objects returned
    • Differents calls to TheMovieDatabase to flesh up the results of search (actor biography, similar movies, etc.)
    • Caching of TheMovieDatabase data using MongoDB to avoid busting the 40 calls per 10 seconds quota
    • Integration of storage for movies reviews
    • API monitoring using NewRelic informing us if any problems occur with the app, so that we can start it back!
    • Expose login through Facebook and Google

    The source code for the API, if you are interested, is available [free of charge!] Just contact Maxime Charron to gain access to it.

    The API itself is hosted here : umovie-team01.herokuapp.com

    Movies review :

    Movies reviews are possible by users. Most recent posts are shown, and live results/preview are displayed when typing a new review.

    Similar movies suggestions :

    Extracted from TheMovieDatabase, a new array gives us the possibility to display similar movies to the one selected.

    Search with autocomplete :

    In the main search bar, autocompletion of search items is provided.

    Gravatar :

    If you email is associated wit a proper Gravatar image, it is displayed in the top menu once logged in.

    And much much more (eye candy!) :

    Many more features were included in this application :

    • Front page : Popular movies and tvShows according to TheMovieDatabase.
    • In the Movies and Tv-Shows pages, a "dashboard" for each genre is displayed, show a selection of movies or tvshows of that particular genre, and uses lazy loading when scrolling down.
    • The dashboards display lots of info, including a preview, for each item on hover (or click in cellphone mode).
    • A nice homemade logo redirects you to the home page.
    • Nice "custom" animation (available here thanks to Shea Lewis), and error message display (available here).

    Using the application (french only)

    Si node et npm sont déjà installés vous pouvez vous rendre au point 4. Si vous avez installé node ou npm en root ne pas oublier de faire les commandes avec sudo.

    1. Installer Node selon votre système d'exploitation
      1. Sur Windows: http://nodejs.org/download/
      2. Sur Mac: http://nodejs.org/download/
      3. Sur Unbuntu 14.04: sudo apt-get update, sudo apt-get install nodejs, sudo apt-get install npm
    2. Vérifier que node est correctement installé: node -v
    3. Vérifier que npm est correctement installé: npm -v
    4. Installer les libraries node via la commande suivante dans répertoire du projet: npm install
    5. Pour lancer l'application, ouvrir un terminal dans le répertoire racine du projet. Écrire ensuite: npm start
    6. Ouvrir chrome et écrire: localhost:8080
